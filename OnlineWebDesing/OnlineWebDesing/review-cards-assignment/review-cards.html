Random HTML and CSS cards

What does the :hover selector do?

The :hover selector is used to select elements when you mouse over them.

what does the div tag do?

The div tag defines a division or a section in an HTML document.

What are the h1-h6 elements used for?

The h1 to h6 HTML elements represent six levels of section headings.

What does the br element do?

The br HTML element produces a line break in text (carriage-return).

What does the p element do?

The p tag defines a paragraph.

What does the b element do?

The b tag specifies bold text without any extra importance.

What does the background-color property do?

The background-color property sets the background color of an element.

What does the font-size property do?

The font-size property changes the size of the text in an element.
